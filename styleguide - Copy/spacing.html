Spacing

Margin and padding

Spacing can be done by adding either margin or padding. both margin and padding has the same predefined spacing. There is 6 different spacing sizes.

HTML <div class="margin-xl"> <div class="margin-l"> <div class="margin-r"> <div class="margin-s"> <div class="margin-xs"> <div class="margin-xxs">
<div class="padding-xl"> <div class="padding-l"> <div class="padding-r"> <div class="padding-s"> <div class="padding-xs"> <div class="padding-xxs">
CSS

margin: var(--margin-xl);

margin: var(--margin-l);

margin: var(--margin-r);

margin: var(--margin-s);

margin: var(--margin-xs);

margin: var(--margin-xxs);


padding: var(--padding-xl);

padding: var(--padding-l);

padding: var(--padding-r);

padding: var(--padding-s);

padding: var(--padding-xs);

padding: var(--padding-xxs);

Individual margin and padding

Individual spacing can be applied to a single side of an element by adding either margin or padding. both margin and padding has the same predefined spacing. again, there is 6 different spacing sizes.

HTML <div class="margin-top-xl"> <div class="margin-top-l"> <div class="margin-top-r"> <div class="margin-top-s"> <div class="margin-top-xs"> <div class="margin-topxxs">
<div class="padding-topxl"> <div class="padding-topl"> <div class="padding-topr"> <div class="padding-tops"> <div class="padding-topxs"> <div class="padding-topxxs">
CSS

margin: var(--margin-top-xl);

margin: var(--margin-top-l);

margin: var(--margin-top-r);

margin: var(--margin-top-s);

margin: var(--margin-top-xs);

margin: var(--margin-top-xxs);


padding: var(--padding-top-xl);

padding: var(--padding-top-l);

padding: var(--padding-top-r);

padding: var(--padding-top-s);

padding: var(--padding-top-xs);

padding: var(--padding-top-xxs);
